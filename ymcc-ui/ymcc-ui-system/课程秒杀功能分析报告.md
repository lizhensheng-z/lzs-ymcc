## 二、API路由说明

### 网关路由配置
前端请求 `http://localhost:10010/ymcc/kill/xxx` 会被网关路由到 `service-kill` 服务，**StripPrefix=2** 会去掉前两个路径 `/ymcc/kill`，所以：

| 前端调用路径 | 网关处理后 | 后端Controller路径 |
|-------------|-----------|-------------------|
| `/kill/killActivity/pagelist` | `/killActivity/pagelist` | `@RequestMapping("/killActivity")` |
| `/kill/killCourse/pagelist` | `/killCourse/pagelist` | `@RequestMapping("/killCourse")` |
| `/kill/killActivity/list` | `/killActivity/list` | `@RequestMapping("/killActivity")` |
| `/kill/killActivity/publish/{id}` | `/killActivity/publish/{id}` | `@RequestMapping("/killActivity")` |
| `/kill/killActivity/delete/{id}` | `/killActivity/delete/{id}` | `@RequestMapping("/killActivity")` |
| `/kill/killCourse/save` | `/killCourse/save` | `@RequestMapping("/killCourse")` |
