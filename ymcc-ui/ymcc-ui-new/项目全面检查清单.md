## 二、详细问题清单

### 🔴 高优先级问题（功能不可用）

#### 1. Table.vue - HTTP 方法大小写错误
**文件**: `src/views/itsource/Table.vue`

**问题**: 第 177 行使用 `this.$http.POST`（大写），应该是 `this.$http.post`（小写）

**代码位置**:
```javascript
// 第177行
this.$http.POST("/system/systemdictionaryitem/save",para).then(...
```

**修复方案**:
```javascript
this.$http.post("/system/systemdictionaryitem/save",para).then(...
```

---

#### 2. Pager.vue - 缺少编辑和删除方法
**文件**: `src/views/itsource/Pager.vue`

**问题**: 模板中引用了 `edit(scope.row)` 和 `del(scope.row)` 方法，但 methods 中未定义

**模板位置**:
```html
<el-button size="small" @click="edit(scope.row)">编辑</el-button>
<el-button type="danger" size="small" @click="del(scope.row)">删除</el-button>
```

**需要添加的方法**:
```javascript
methods: {
    // ... 现有方法 ...

    edit(row) {
        this.addForm = Object.assign({}, row);
        this.addFormVisible = true;
    },

    del(row) {
        this.$confirm('确认删除吗？', '提示', {
            type: 'warning'
        }).then(() => {
            this.$http.delete(`/pager/pager/${row.id}`).then(result => {
                if (result.data.success) {
                    this.$message.success('删除成功');
                    this.getTableData();
                }
            });
        });
    }
}
```
