## 二、详细流程分析

### 步骤1：创建秒杀活动

**前端**：`ymcc-ui-system/src/views/itsource/KillActivity.vue`

**API**：`POST /kill/killActivity/save`

**后端**：`KillActivityController.saveOrUpdate()`

**数据库**：`t_kill_activity`

```sql
INSERT INTO t_kill_activity (name, start_time, end_time, time_str, publish_status)
VALUES ('活动名称', '2026-03-31 10:00:00', '2026-03-31 12:00:00', '10:00-12:00', 0);
```

**状态**：`publish_status = 0`（待发布）

### 步骤2：添加秒杀课程

**前端**：`ymcc-ui-system/src/views/itsource/Course.vue` → 点击"加入秒杀"

**API**：`POST /kill/killCourse/save`

**请求参数**：
```json
{
  "courseId": 123,
  "courseName": "Vue.js实战课程",
  "coursePic": "http://xxx/pic.jpg",
```

**后端**：`KillCourseController.saveOrUpdate()` → `KillCourseServiceImpl.addKillCourse()`

**代码逻辑**：
```java
public void addKillCourse(KillCourse killCourse) {
    // 1. 检查是否已存在
    // 2. 查询活动信息
    // 3. 设置时间信息（从活动复制）
```

**⚠️ 注意**：此时只存入数据库，**不存入Redis**，用户端查不到。

### 步骤3：发布秒杀活动（关键步骤）

**前端**：`ymcc-ui-system/src/views/itsource/KillActivity.vue` → 点击"发布"

**API**：`POST /kill/killActivity/publish/{activityId}`

**后端**：`KillActivityController.publish()` → `KillActivityServiceImpl.publish()`

**代码逻辑**：
```java
public void publish(Long activityId) {
    // 1. 查询活动
    KillActivity killActivity = selectById(activityId);

    // 2. 查询活动下的所有秒杀课程（从DB查）
    List<KillCourse> killCourses = killCourseService.selectList(wrapper);

    // 3. 更新活动状态
    killActivity.setPublishStatus(1);  // 已发布
    updateById(killActivity);

    // 4. 遍历课程，存入Redis
    for (KillCourse killCourse : killCourses) {
        // 4.1 更新课程状态
        killCourse.setPublishStatus(1);
        killCourseService.updateById(killCourse);

        // 4.2 设置库存信号量
        RSemaphore semaphore = redissonClient.getSemaphore(killCourse.getId().toString());
        semaphore.trySetPermits(killCourse.getKillCount());

        // 4.3 存入Redis Hash
        String key = "activity:" + killActivity.getId();
        redisTemplate.opsForHash().put(key, killCourse.getId().toString(), killCourse);
    }
}
```

**Redis存储结构**：
```
Key: activity:1
Type: Hash
Value: {
  "101": KillCourse对象JSON,
  "102": KillCourse对象JSON
}

Key: 101 (课程ID)
Type: Semaphore
Value: 100 (库存数量)
```

### 步骤4：用户查询秒杀列表

**前端**：`ymcc-ui-new/src/views/kill/KillList.vue`

**API**：`GET /kill/killCourse/online/all`

**后端**：`KillCourseController.onlineAll()` → `KillCourseServiceImpl.onlineAll()`

**代码逻辑**：
```java
public List<KillCourse> onlineAll() {
    // 1. 从Redis查询所有活动key
    Set<Object> keys = redisTemplate.keys("activity:*");

    // 2. 遍历获取商品
    List<KillCourse> killCourses = new ArrayList<>();
    for (Object key : keys) {
        List values = redisTemplate.opsForHash().values(key);
        killCourses.addAll(values);
    }
    return killCourses;
}
```

**⚠️ 问题**：如果活动未发布，Redis中没有数据，返回空列表。

### 步骤5：用户查看秒杀详情

**前端**：`ymcc-ui-new/src/views/kill/KillDetail.vue`

**API**：`GET /kill/killCourse/online/one/{killId}/{activityId}`

**后端**：`KillCourseController.onlineOne()`

**代码逻辑**：
```java
public KillCourse onlineOne(Long killId, Long activityId) {
    String key = "activity:" + activityId;
    return (KillCourse) redisTemplate.opsForHash().get(key, killId.toString());
}
```

### 步骤6：用户执行秒杀

**前端**：点击"立即秒杀"

**API**：`POST /kill/killCourse/kill`

**请求参数**：
```json
{
  "killCourseId": 101,
  "activityId": 1
}
```

**后端**：`KillCourseController.kill()` → `KillCourseServiceImpl.kill()`

**代码逻辑**：
```java
public String kill(KillParamDto dto) {
    // 1. 获取用户ID
    Long loginId = LoginContext.getLogin().getId();

    // 2. 检查是否重复购买
    String repeatKey = loginId + ":" + dto.getKillCourseId();
    Object repeat = redisTemplate.opsForValue().get(repeatKey);
    if (repeat != null) throw new RuntimeException("已购买过");

    // 3. 查询秒杀课程
    KillCourse killCourse = onlineOne(dto.getKillCourseId(), dto.getActivityId());

    // 4. 校验秒杀时间
    if (!killCourse.isKilling()) throw new RuntimeException("不在秒杀时间");

    // 5. 扣减库存（信号量）
    RSemaphore semaphore = redissonClient.getSemaphore(killCourse.getId().toString());
    boolean success = semaphore.tryAcquire(1);
    if (!success) throw new RuntimeException("库存不足");

    // 6. 生成预订单
    String orderNo = CodeGenerateUtils.generateOrderSn(loginId);
    PreOrderDto preOrder = new PreOrderDto(orderNo, killCourse.getKillPrice(), 1, loginId, killCourse.getCourseId());

    // 7. 存入Redis
    redisTemplate.opsForValue().set(orderNo, preOrder);
    redisTemplate.opsForValue().set(repeatKey, orderNo);

    return orderNo;
}
```

## 五、操作指南

### 管理员操作步骤

1. **创建秒杀活动**
   - 进入"秒杀中心" → "秒杀活动"
   - 点击"新增"，填写活动名称、开始时间、结束时间
   - 点击"保存"

2. **添加秒杀课程**
   - 进入"课程管理"
   - 选择课程，点击"加入秒杀"
   - 选择秒杀活动，填写秒杀价格、秒杀数量
   - 点击"提交"

3. **发布秒杀活动** ⚠️ 关键步骤
   - 进入"秒杀中心" → "秒杀活动"
   - 找到对应活动，点击"发布"
   - 此时数据才会存入Redis

### 用户操作步骤

1. 访问首页，查看"限时秒杀"入口
2. 或直接访问 `/kill` 秒杀列表页
3. 点击课程查看详情
4. 点击"立即秒杀"参与秒杀

## 六、常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 秒杀列表为空 | 活动未发布 | 点击"发布"按钮 |
| 秒杀列表为空 | 活动时间已过 | 创建新的活动 |
| 秒杀按钮禁用 | 不在秒杀时间 | 等待秒杀开始 |
| 秒杀失败 | 库存不足 | 增加库存 |
| 秒杀失败 | 已购买过 | 每人限购1件 |
| Redis无数据 | 服务未启动 | 启动Redis服务 |
